Project documents for my Devmountain Foundations capstone project.

Planning documents are at https://github.com/johager/foundations-capstone-docs

After cloning the repo, execute the following:

npm init -y
npm i express
npm i bcryptjs
npm i dotenv
npm i sequelize
npm i pg pg-hstore


The .env file should include
PORT - port for the server
CONNECTION_STRING - URI for the PostgreSQL database
